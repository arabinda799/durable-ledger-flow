# AI Disclosure

This document details the usage of Artificial Intelligence (AI) tools during the development of the Durable Game Economy Service.

---

## Tools Used

- **ChatGPT**
  - **Purpose**: Used for high-level database schema design, initial research into TypeORM pessimistic write locking strategies, and outlining test case structures.
  - **Utilization**: Approximately 15% of the overall architecture decisions and structural templates were derived from initial ChatGPT prompts.

- **Antigravity (IDE Pair-Programming Agent)**
  - **Purpose**: Used for writing E2E test suites, database seeder files, and markdown documentation files (no production application code).
  - **Utilization**: Assisted in compiling the test cases, bootstrapping seed data checks, and formatting project documentation.
